# Statistical-Analysis-R-HE
Higher Education Data Analysis in R

Project Overview
This repository contains a statistical analysis conducted as part of my academic coursework, focused on exploring trends and patterns within Higher Education datasets. The project demonstrates the application of R programming for data manipulation and statistical reasoning.

Key Technical Tasks
Data Cleaning & Preprocessing: Managing datasets in R to ensure consistency and readiness for analysis.
Descriptive Statistics: Calculating key metrics to summarize educational trends.
Data Visualization: Creating plots and charts to effectively communicate statistical findings.

Interpretation: Analyzing the outputs to draw meaningful conclusions about educational dynamics.

Tools Used

Language: R
Environment: RStudio
